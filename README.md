## Guia de Estudo: As Carreiras de Dados

### Resumo das Funções (Analogia do Restaurante 🍳)
Para facilitar o entendimento de como estes profissionais trabalham em conjunto, podemos usar a metáfora de um restaurante de alta gastronomia:

* *O Engenheiro de Dados (O Fornecedor):* É quem busca os melhores ingredientes (dados), limpa-os, organiza-os nas prateleiras e garante que a cozinha tem água e gás a funcionar. Sem ele, os outros não têm matéria-prima.
* *O Analista de Dados (O Garçom/Maitre):* Observa o salão, conversa com os clientes e percebe o que eles mais gostam de comer. Ele traduz os comportamentos para que os gestores tomem decisões melhores.
* *O Cientista de Dados (O Chef Executivo):* Pega nos ingredientes organizados e faz experiências. Cria receitas complexas (modelos de IA) e tenta prever qual será o prato de maior sucesso na próxima estação do ano.
* *O Engenheiro de Machine Learning (O Engenheiro de Alimentos):* Pega na receita perfeita criada pelo Chef e faz com que ela seja produzida em larga escala, numa fábrica, sem perder o sabor e a qualidade.

###  Glossário do Iniciante em Dados
* *Modelos Preditivos:* Modelos matemáticos que usam dados históricos para tentar prever comportamentos ou resultados futuros.
* *Machine Learning:* Subcampo da IA focado no desenvolvimento de sistemas que aprendem e melhoram o seu desempenho sozinhos através de dados.
* *Dados Estruturados:* Informação altamente organizada (geralmente em tabelas de linhas e colunas) e fácil de pesquisar.
